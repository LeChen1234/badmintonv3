启动前端步骤
=============

1. 安装 Node.js（若尚未安装）
   - 打开 https://nodejs.org/
   - 下载并安装 LTS 版本
   - 安装时务必勾选 “Add to PATH”

2. 重新打开终端
   - 关闭当前 Cursor/终端窗口，再开一个新的（或重启 Cursor）
   - 这样新终端才会识别 npm

3. 启动前端（任选一种方式）

   方式 A：双击运行
   - 在项目根目录 d:\badminton 下双击 run_frontend.bat

   方式 B：命令行
   - 打开 PowerShell 或 CMD，执行：
     cd d:\badminton\frontend
     npm install
     npm run dev

4. 浏览器访问
   - 打开 http://localhost:3000

若仍提示 “npm 不是内部或外部命令”，说明 Node 未加入 PATH，请重新安装 Node.js 并勾选 “Add to PATH”，或手动将 Node 安装目录（如 C:\Program Files\nodejs）加入系统环境变量 PATH。
