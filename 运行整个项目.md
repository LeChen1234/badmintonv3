# 运行整个项目

本地运行需已安装 **Python 3.10+**、**Node.js 18+**，以及 **PostgreSQL 16**、**Redis**。

## 首次：安装依赖

在 PowerShell 中执行（依赖会装到 D 盘项目下）：

```powershell
cd d:\badminton
.\scripts\install_and_run.ps1
```

该脚本会：创建 `.venv`、安装 Python 依赖、安装前端依赖、生成 `.env`（若无）、执行数据库迁移。

## 启动后端和前端

**方法 A：双击运行**

1. 双击 **`run_backend.bat`**，保持窗口不关。
2. 再双击 **`run_frontend.bat`**，保持窗口不关。

**方法 B：命令行**

```powershell
# 终端 1 - 后端
cd d:\badminton\backend
..\.venv\Scripts\python.exe -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

# 终端 2 - 前端
cd d:\badminton\frontend
npm run dev
```

然后访问：

- 管理前端：http://localhost:3000  
- 后端健康检查：http://localhost:8000/api/health

## 常见问题

- **后端启动报 “No module named uvicorn”**：先执行一次 `.\scripts\install_and_run.ps1` 或  
  `d:\badminton\.venv\Scripts\python.exe -m pip install -r d:\badminton\backend\requirements.txt`。
- **前端报 “npm 不是内部或外部命令”**：请安装 Node.js 并确保安装时勾选 “Add to PATH”。
- **后端连不上数据库**：确保 PostgreSQL、Redis 已在本机启动，`.env` 中 `POSTGRES_HOST=localhost`、`REDIS_HOST=localhost`。
- **后端报 UnicodeDecodeError**：请将 `.env` 用 **UTF-8** 编码保存（不要用 GBK）；若含中文注释，可用 VS Code / 记事本另存为 UTF-8。
