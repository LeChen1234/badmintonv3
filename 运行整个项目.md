# 运行整个项目

本地运行需安装：

- Python 3.9+（推荐 3.10+）
- Node.js 18+（含 npm）

本项目当前默认使用 SQLite，不依赖 PostgreSQL/Redis 才能启动管理前后端。

## 一键运行（推荐）

在项目根目录执行：

```powershell
.\run.bat
```

或双击 `run_all.bat`（内部会调用 `run.bat`）。

脚本会：

1. 创建/复用 `.venv`
2. 安装后端依赖 `backend/requirements.txt`
3. 安装前端依赖 `frontend/node_modules`（若缺失）
4. 启动后端（8000）与前端（3000）

## 分开启动

```powershell
# 后端
.\run_backend.bat

# 前端
.\run_frontend.bat
```

## 访问地址

- 前端：http://localhost:3000
- 后端健康检查：http://localhost:8000/api/health

## 平台初始化（可选）

```powershell
.\.venv\Scripts\python.exe scripts\init_platform.py
```

如果你还没启动 Label Studio，可以先跳过 LS 初始化：

```powershell
.\.venv\Scripts\python.exe scripts\init_platform.py --skip-ls
```

## 常见问题

- `Python not found on PATH`：安装 Python 并确保加入 PATH，或先手动创建 `.venv`
- `npm not found`：安装 Node.js 并勾选 `Add to PATH`
- `alembic upgrade` 报表已存在：脚本会自动尝试 `alembic stamp head`
